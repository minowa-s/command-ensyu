#誰でも簡単にBMIを測定できる！
##身長と体重を入力するだけで簡単にBMIを測定できます。
##一回の実行で2人分測定出来るので家族での利用や昔の自分との比較もしやすいです。
###使い方
1.1人目の身長をメートル(m)単位で入力します。
1.2人目の体重をキログラム(kg)単位で入力します。
1.2人目の身長をメートル(m)単位で入力します。
1.2人目の体重をメートル(m)単位で入力します。
1.結果が表示されます。
